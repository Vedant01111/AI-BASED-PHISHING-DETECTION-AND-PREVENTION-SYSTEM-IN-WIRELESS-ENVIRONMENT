About the project

Phishing detection using machine learning is a critical part of modern cybersecurity. iSecure analyses URLs, email headers, and email bodies to classify them as legitimate or malicious in real time, helping users avoid credential theft and financial fraud.

The URL classifier extracts 30 lexical, network, and content features (domain age, HTTPS usage, anchor ratios, WHOIS data, traffic rank, and more) and feeds them to a pre-trained gradient boosting model. The email pipeline pairs SPF/DKIM/DMARC heuristics with a per-user TF-IDF + Random Forest classifier trained on the user's own Gmail history.

Models are continuously improvable: the Gmail trainer rebuilds your personal classifier from the latest 200 messages plus a curated phishing corpus, so detection adapts to your context.
